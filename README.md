# McDonald's Website Redesign

## Overview

This project is a redesign of the McDonald's website using only HTML and CSS. The goal was to create a modern, user-friendly, and visually appealing website that enhances the user experience while maintaining the brand's identity.

## Features

- **Responsive Design**: The website is fully responsive and works seamlessly on various devices, including desktops, tablets, and mobile phones.
- **Clean and Modern UI**: The design focuses on simplicity and modern aesthetics to provide a pleasant browsing experience.
- **Improved Navigation**: A well-structured and intuitive navigation system makes it easy for users to find the information they need.
- **Brand Consistency**: The redesign maintains McDonald's brand colors, fonts, and overall style to ensure brand consistency.

## Technologies Used

- **HTML**: Used for structuring the content and layout of the website.
- **CSS**: Used for styling the website, including layout, colors, fonts, and responsiveness.

## Getting Started

To view the redesigned McDonald's website, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/MercySpectures/Mcdonalds-ReImagine.git
   ```

2. **Navigate to the project directory**:
   ```bash
   cd Mcdonalds-ReImagine
   ```

3. **Open `index.html` in your web browser**:
   ```bash
   open index.html
   ```
   or simply double-click the `index.html` file to open it in your default web browser.
